About Smart TV Player

Developer: Sultan Arabi

Version: 1.0.5

Features:

  • Multi-platform support (Desktop, Mobile, Android TV)
  • Keyboard, Mouse, Remote, and Touch controls
  • Channel search functionality
  • PIP mode support
  • Full-screen mode
  • Volume control
  • Channel navigation

Controls:

  • Arrow keys: Navigate channels
  • Enter: Select channel
  • Space: Play/Pause
  • M: Mute/Unmute
  • F: Toggle fullscreen
  • P: Toggle PIP mode
